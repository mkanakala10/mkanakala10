## Me

I am a software engineer wrapping up my Bachelor of Science in Computer Science with a Data Science minor at the University of Wisconsin–Madison, where I will also be beginning my Professional Master of Science in Computer Science this fall. 

My engineering focus bridges AI, high-performance data systems, and scalable full-stack development. Recently, I've been building event-driven serverless architectures, optimizing LLM content pipelines, and developing fault-tolerant real-time data streams. I also enjoy exploring high-performance computing concepts, GPU architecture optimizations, and remote sensing data analysis.

## Experience & Projects

*   **AI Engineering @ Unique Minds:** Built an event-driven serverless backend using Firebase and optimized a Gemini content generation pipeline, cutting average response times by 60%.
*   **Software Engineering @ DiClano:** Designed a full-stack, RBAC-enabled SaaS platform using FastAPI, React, and PostgreSQL for over 1,000 users, maintaining a P99 latency of under 100ms.
*   **Software Engineering @ VestaLabs:** Architected a serverless video processing pipeline integrating GCP Storage, BigQuery, and Vertex AI for automated metadata extraction.
*   **Real-Time Data Pipeline:** Developed a fault-tolerant pipeline in Java using Apache Kafka, Spark Streaming, and Cassandra to process over 10,000 events per second.
*   **Predictive Sports Analytics:** Trained machine learning models (Random Forest, Logistic Regression) to simulate and predict NCAA basketball outcomes, achieving 79.4% accuracy across 2,000+ games.
